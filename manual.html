Manual Técnico del Proyecto Web

1. Estructura del sitio

  • El archivo principal es index.html, el cual contiene un menú lateral y un área de contenido con uso de <iframe>.
  • El menú incluye enlaces a las secciones requeridas, las cuales se cargan dentro del mismo sitio sin recargar la página completa.

2. Archivos creados

  • index.html: diseño principal con estructura moderna.
  • hoja_vida.html: datos personales y académicos, junto con una foto del alumno.
  • notas.html: tabla con nombres de cursos y calificaciones.
  • galeria.html: galería con 10 imágenes, presentada en formato visual atractivo.
  • video.html: video de una canción favorita incrustado desde YouTube.
  • juego.html: juego interactivo desarrollado con JavaScript (Snake/laberinto).
  • manual.html: este manual que explica cómo se construyó todo el sitio.

3. Herramientas utilizadas

  • Visual Studio Code: para escribir y organizar el código.
  • HTML y CSS: para la estructura y el diseño visual.
  • JavaScript: para la lógica de los juegos y la interacción.
  • Canva / Edición de imágenes: para redimensionar o preparar contenido visual (opcional).

4. Estilo visual

  • Se usó una paleta de colores sobria y moderna (tonos azul y gris oscuro).
  • El diseño es limpio y responsivo, se adapta bien a pantallas medianas.
  • Se priorizó claridad, accesibilidad y facilidad de navegación.

5. Pruebas y validación

  • Se probó el sitio en navegadores como Google Chrome y Microsoft Edge.
  • Todos los enlaces del menú fueron verificados manualmente.
  • Los juegos y videos funcionan correctamente dentro del iframe.
Nota: Este sitio fue desarrollado exclusivamente como parte de la práctica académica de la asignatura “Introducción a los sistemas de cómputo”.
